# 🧾 Expense Tracker App 💰

Expense Tracker App is a full-stack web application that helps users manage and visualize their expenses efficiently. It features user authentication, dynamic expense tracking, and a clean, responsive UI built with React and Tailwind CSS. The backend is powered by Express.js, Node.js, and MongoDB, providing robust data management and API support.

---

## Features ✨

- 🔐 **User Authentication:** Secure login and signup functionality to manage user sessions.
- 📱 **Responsive Design:** Fully responsive UI that works seamlessly across all device sizes.
- 🛠️ **Technology Stack:** Built with React and Tailwind CSS on the frontend, and Express.js, Node.js, and MongoDB on the backend.
- 📊 **Visual Expense Analytics:** Interactive Pie Chart visualization to display expenses by category.
- 🛒 **Expense Categories:** Supports 7 distinct expense categories: Grocery, Vehicle, Shopping, Travel, Food, Fun, and Other.
- 🎨 **Color-Coded Chart:** Each expense category is represented with a unique color in the pie chart for easy distinction.
- ⚡ **Live Updates:** Expense data and visualizations update live as new expenses are added or removed.

---

## Usage 🎯

- Register a new account or login with existing credentials.
- Add expenses by selecting a category and entering the amount.
- View your expenses summarized in a colorful pie chart by category.
- The app updates all data live without requiring page reloads.
- Logout when finished.

---

## Technologies Used 🛠️

- **Frontend:** React ⚛️, Tailwind CSS 🎨, react-chartjs-2 📈 (Chart.js)
- **Backend:** Express.js 🚂, Node.js 🟢
- **Database:** MongoDB 🍃
- **Other Libraries:** react-hot-toast 🔥, react-router-dom 🔀, react-icons 🎨
